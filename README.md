
# 🎬 Cinema Management System (JavaFX)

A desktop application built with **JavaFX** for managing a cinema system.  
The system allows managing movies, halls, schedules.

---

## 🛠 Technologies Used
- Java  
- JavaFX  
- NetBeans IDE  
- JDBC  

---

## 🚀 How to Run the Project

1. Install **NetBeans** and **JDK**.
3. Start **Apache** ز

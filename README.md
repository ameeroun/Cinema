
# 🎬 Cinema Management System (JavaFX)

A desktop application built with **JavaFX** for managing a cinema system.  
The system allows managing movies, halls, schedules, and reservations using a MySQL database (XAMPP).

---

## 🛠 Technologies Used
- Java  
- JavaFX  
- MySQL (XAMPP)  
- NetBeans IDE  
- JDBC  

---
